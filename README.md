# 🐍 Snake Game — React Native

A classic Snake game built with React Native and Expo. Navigate the snake around the board using swipe gestures to eat food and grow longer without hitting the walls.

## Built With
- React Native + Expo
- TypeScript
- React Native Gesture Handler

## Getting Started
```bash
git clone https://github.com/J3450/Snake-Game.git
cd Snake
npm install
npx expo start
```
